### 📝 이번 버전에서 업그레이드된 사항 (What's New)
**App Store Connect의 "이번 버전에서 업그레이드된 사항" 필드에 입력**

```
✨ 새로운 기능

📱 실시간 푸시 알림 시스템
• 앱을 완전히 종료한 상태에서도 알림을 받을 수 있습니다
• 서버 기반 알림으로 더욱 안정적인 알림 수신이 가능합니다

💬 채팅 알림 개선
• 채팅 메시지가 도착하면 즉시 알림을 받습니다
• 모임 제목과 발신자 정보가 포함된 알림으로 더욱 명확합니다

👥 모임 관련 알림
• 모임이 취소되면 참여자에게 즉시 알림이 전송됩니다
• 새로운 참여자가 모임에 참여하면 주최자에게 알림이 전송됩니다

❤️ 커뮤니티 알림
• 게시글에 좋아요를 받으면 실시간으로 알림을 받습니다
• 댓글이 달리면 즉시 알림을 받아 소통을 놓치지 않습니다

⚙️ 알림 설정 개선
• 알림 설정이 모든 기기에서 동기화됩니다
• 원하는 알림만 선택적으로 받을 수 있습니다

🎨 UI 개선
• 채팅 화면의 말풍선 디자인이 더욱 깔끔해졌습니다
• 알림 화면에서 게시글 조회가 더욱 원활해졌습니다

🐛 버그 수정
• 알림 화면에서 게시글을 찾을 수 없는 문제가 해결되었습니다
• 알림 권한 요청이 시뮬레이터에서도 정상적으로 작동합니다
```

## 📝 참고 사항

### App Store Connect 입력 가이드

1. **프로모션 텍스트**
   - 위치: App Store Connect > 앱 정보 > 프로모션 텍스트
   - 최대 170자
   - 앱 설명 위에 표시됨
   - 마케팅에 활용되는 중요한 텍스트

2. **이번 버전에서 업그레이드된 사항**
   - 위치: App Store Connect > 버전 정보 > 이번 버전에서 업그레이드된 사항
   - 최대 4,000자
   - 사용자가 앱 업데이트 시 가장 먼저 보는 내용
   - 명확하고 이해하기 쉬운 문구 사용

3. **권장 사항**
   - 이모지 사용으로 가독성 향상
   - 사용자 관점에서 작성
   - 기술적 용어보다는 사용자 친화적 표현 사용
   - 주요 기능을 먼저 언급
