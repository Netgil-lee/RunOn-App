# Apple App Review 답변 제출 가이드

## 📍 답변 제출 위치

Apple App Review에 답변을 제출하는 방법은 두 가지가 있습니다:

---

## 방법 1: Resolution Center에서 답변 (권장)

### 단계별 가이드

1. **App Store Connect 접속**
   - https://appstoreconnect.apple.com 접속
   - Apple Developer 계정으로 로그인

2. **앱 선택**
   - "My Apps" 클릭
   - RunOn 앱 선택

3. **App Review 섹션으로 이동**
   - 왼쪽 메뉴에서 "App Review" 클릭
   - 또는 상단 탭에서 "App Review" 선택

4. **Resolution Center 확인**
   - "Resolution Center" 섹션 확인
   - "Guideline 2.1 - Information Needed" 메시지 확인

5. **답변 작성 및 제출**
   - 메시지 하단의 "Reply" 또는 "답변" 버튼 클릭
   - `APPLE_REVIEW_RESPONSE.md` 파일의 내용을 복사하여 붙여넣기
   - "Submit" 또는 "제출" 클릭

### 답변 형식

```
안녕하세요,

Guideline 2.1 요구사항에 대한 답변을 제출드립니다.

[여기에 APPLE_REVIEW_RESPONSE.md의 내용을 붙여넣기]

감사합니다.
RunOn 개발팀
```

---

## 방법 2: App Review Information에 추가

### 단계별 가이드

1. **App Store Connect 접속**
   - https://appstoreconnect.apple.com 접속
   - Apple Developer 계정으로 로그인

2. **앱 선택**
   - "My Apps" 클릭
   - RunOn 앱 선택

3. **App Information 섹션으로 이동**
   - 왼쪽 메뉴에서 "App Information" 클릭
   - 또는 상단 탭에서 "App Information" 선택

4. **App Review Information 수정**
   - "App Review Information" 섹션 찾기
   - "Notes" 또는 "Review Notes" 필드에 답변 내용 추가
   - 또는 "Contact Information" 섹션에 연락처 확인

5. **저장 및 재제출**
   - "Save" 클릭
   - 앱을 다시 제출 (Submit for Review)

## 질문 1: 부적절한 콘텐츠를 필터링하는 방법이 있습니까?

답변: 예, 있습니다.

RunOn 앱은 실시간 콘텐츠 필터링 시스템을 구현하고 있습니다.

구현 위치:
- 게시글 작성 화면 (PostCreateScreen)
- 댓글 작성 화면 (PostDetailScreen)

작동 방식:
1. Level 1 (심각) - 즉시 차단: 욕설, 성적 표현, 폭력, 혐오 표현 등이 포함된 콘텐츠는 게시 전에 자동으로 차단됩니다.
2. Level 2 (경미) - 경고 후 선택: 경미한 부적절한 표현은 경고 메시지와 함께 사용자에게 선택권을 제공합니다.

확인 방법:
앱 실행 → 커뮤니티 탭 → 게시글 작성 → 부적절한 단어(예: "시발") 입력 시도 → 즉시 차단 메시지 확인

## 질문 2: 사용자가 부적절한 콘텐츠를 신고할 수 있는 메커니즘이 있습니까?

답변: 예, 있습니다.

RunOn 앱은 게시글, 댓글, 사용자에 대한 신고 기능을 제공합니다.

구현 위치:
1. 게시글 신고: 게시글 상세 화면 헤더 우측 경고 아이콘(⚠️)
2. 댓글 신고: 댓글 길게 누르기 (Long Press)
3. 사용자 신고: 사용자 프로필 화면

신고 사유:
- 스팸/광고
- 부적절한 콘텐츠
- 혐오 표현
- 성적 콘텐츠
- 폭력/위협
- 기타

확인 방법:
앱 실행 → 커뮤니티 탭 → 게시글 클릭 → 헤더 우측 경고 아이콘(⚠️) 클릭 → 신고 사유 선택 → 제출

## 질문 3: 사용자가 악의적인 사용자를 차단할 수 있는 메커니즘이 있습니까?

답변: 예, 있습니다.

RunOn 앱은 사용자 차단 기능을 제공합니다.

구현 위치:
- 사용자 프로필 화면 (ParticipantScreen)
- 차단 관리 화면 (BlacklistManagementScreen)

작동 방식:
1. 사용자 프로필 화면에서 "차단" 버튼 클릭
2. 양방향 차단 처리 (차단한 사용자와 차단당한 사용자 모두 서로를 차단)
3. 차단된 사용자와 함께 있는 모든 모임에서 자동 탈퇴
4. 차단된 사용자의 게시글/댓글이 표시되지 않음

차단 제한:
- 최대 3명까지 차단 가능
- 차단 해제는 설정 화면에서 가능

확인 방법:
앱 실행 → 모임 상세 화면 → 참여자 목록 → 사용자 프로필 클릭 → "차단" 버튼 클릭
