# Health Connect 권한 사용 목적 선언

## android.permission.health.READ_EXERCISE 권한 사용 목적

RunOn 앱은 러닝 모임 및 커뮤니티 서비스를 제공하는 앱으로, 사용자가 참여한 러닝 모임과 연동하여 실제 운동 기록을 조회하고 공유하는 기능을 제공합니다.

### 권한 사용 목적

1. **러닝 모임과 운동 기록 연동**
   - 사용자가 참여한 러닝 모임의 시간대와 일치하는 운동 기록을 Health Connect에서 조회합니다.
   - 모임 시간 기준 ±30분 범위 내의 러닝 운동 기록을 자동으로 찾아 매칭합니다.

2. **운동 기록 조회 및 표시**
   - 조회된 운동 기록의 거리, 운동 시간, 칼로리, 평균 페이스 등의 정보를 사용자에게 표시합니다.
   - 사용자가 참여한 모임의 실제 운동 성과를 확인할 수 있도록 합니다.

3. **운동 기록 공유 기능**
   - 사용자가 원하는 경우, 조회된 운동 기록을 카드 형태로 생성하여 공유할 수 있습니다.
   - 운동 기록은 사용자의 명시적 요청에 의해서만 조회되며, 자동으로 수집하거나 저장하지 않습니다.

### 데이터 사용 방식

- **읽기 전용 접근**: 앱은 운동 기록을 읽기만 하며, Health Connect에 데이터를 쓰거나 수정하지 않습니다.
- **사용자 주도 조회**: 운동 기록은 사용자가 명시적으로 "운동 기록 조회" 또는 "공유" 기능을 사용할 때만 조회됩니다.
- **시간 기반 필터링**: 러닝 모임의 시작 시간을 기준으로 해당 시간대의 운동 기록만 조회합니다.
- **데이터 매칭**: 조회된 운동 기록은 해당 모임과 자동으로 연동되어 표시됩니다.

### 사용자 이점

- 러닝 모임 참여 후 실제 운동 기록을 쉽게 확인할 수 있습니다.
- 운동 기록을 시각적으로 공유하여 커뮤니티와 소통할 수 있습니다.
- 여러 건강 앱의 데이터를 Health Connect를 통해 통합 관리할 수 있습니다.

### 개인정보 보호

- 운동 기록은 사용자의 기기에서만 처리되며, 서버로 전송되지 않습니다.
- 사용자는 언제든지 앱 설정에서 Health Connect 권한을 철회할 수 있습니다.
- 앱은 운동 기록을 로컬에서만 사용하며, 제3자와 공유하지 않습니다.
