# Garmin Connect 개발자 프로그램 활용 계획 (답변용)

**질문:** Garmin Connect 개발자 프로그램을 어떻게 활용하실 계획이신가요?

---

## 답변 (본문)

RunOn은 러닝 모임 매칭과 커뮤니티를 제공하는 모바일 앱입니다. Garmin Connect 개발자 프로그램을 아래와 같이 활용할 계획입니다.

**1. 러닝 활동 데이터 연동**  
Garmin Connect API를 통해 사용자의 러닝·워크아웃 활동(거리, 시간, 페이스, 경로 등)을 안전하게 읽어오고, RunOn 앱 내 대시보드·프로필·통계에 반영할 예정입니다. iOS에서는 HealthKit, Android 및 웨어러블 사용자층을 위해 Garmin 데이터를 보완 데이터 소스로 활용합니다.

**2. 러닝 기록 공유 및 커뮤니티 기능**  
가져온 활동 데이터를 바탕으로, 사용자가 참여한 러닝 모임과 연결된 기록을 커뮤니티·피드에 공유할 수 있도록 합니다. 공유 시 Garmin 기기/앱에서 기록한 러닝임을 표시하고, 필요 시 활동 상세 페이지로 연결하는 등 출처를 명확히 하겠습니다.

**3. 사용자 경험 통일**  
Garmin 워치를 주로 사용하는 사용자도 RunOn에서 별도 입력 없이 자신의 러닝 이력을 확인하고, 모임 참여 전후 기록을 한 곳에서 관리할 수 있도록 합니다. 동의 절차와 데이터 사용 범위는 앱 내 개인정보처리방침 및 이용약관에 명시하겠습니다.

**4. 기술적 준수 사항**  
Garmin Connect API 이용 약관 및 개발자 가이드를 준수하고, OAuth 등 인증·동의 절차를 거쳐 최소한의 데이터만 요청·저장하며, 사용자가 연동 해제 및 데이터 삭제를 요청할 수 있도록 구현할 예정입니다.

요약하면, Garmin Connect는 RunOn에서 **러닝 활동 데이터를 읽어와 대시보드·통계·커뮤니티 공유에 활용**하기 위한 목적으로만 사용하며, 사용자 동의와 개인정보 보호를 우선하겠습니다.

---

## 요약 (짧은 버전, 글자 수 제한 시)

RunOn은 러닝 모임·커뮤니티 앱입니다. Garmin Connect API로 사용자의 러닝 활동(거리, 시간, 페이스, 경로)을 읽어와 앱 내 대시보드·통계·러닝 기록 공유에 활용할 계획입니다. Garmin 워치 사용자도 RunOn에서 별도 입력 없이 기록을 확인·공유할 수 있도록 하며, OAuth·개인정보처리방침·연동 해제 기능을 준수해 구현하겠습니다.
