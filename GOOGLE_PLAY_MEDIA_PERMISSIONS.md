# 구글 플레이 콘솔 미디어 권한 설명

## 📱 READ_MEDIA_IMAGES 권한 설명

**구글 플레이 콘솔에 입력할 내용 (최대 250자):**

```
이 앱은 사용자 프로필 사진 업로드 및 커뮤니티 게시글에 이미지를 첨부하기 위해 갤러리의 이미지에 접근합니다. 사용자가 직접 선택한 이미지만 사용하며, 갤러리의 다른 이미지에는 접근하지 않습니다.
```

**문자 수**: 약 90자

---

## 🎥 READ_MEDIA_VIDEO 권한 설명

**구글 플레이 콘솔에 입력할 내용 (최대 250자):**

```
이 앱은 향후 커뮤니티 게시글에 동영상을 첨부하는 기능을 제공하기 위해 갤러리의 동영상에 접근할 수 있는 권한을 요청합니다. 현재는 이미지만 사용하며, 동영상 기능은 추후 업데이트에서 제공될 예정입니다.
```

**문자 수**: 약 100자

---

## 📋 대안: 더 간결한 버전

### READ_MEDIA_IMAGES (간결 버전)

```
프로필 사진 업로드 및 게시글 이미지 첨부를 위해 갤러리의 이미지에 접근합니다. 사용자가 직접 선택한 이미지만 사용합니다.
```

**문자 수**: 약 50자

---

### READ_MEDIA_VIDEO (간결 버전)

```
향후 게시글에 동영상을 첨부하는 기능을 위해 갤러리의 동영상에 접근할 수 있는 권한을 요청합니다. 현재는 사용하지 않으며 추후 업데이트에서 제공될 예정입니다.
```

**문자 수**: 약 70자

## 📝 권한 사용 상세 정보

### READ_MEDIA_IMAGES 사용 위치:
1. **프로필 화면** (`screens/ProfileScreen.js`)
   - 프로필 사진 업로드
   - 갤러리에서 이미지 선택
   - 카메라로 사진 촬영

2. **게시글 작성 화면** (`screens/PostCreateScreen.js`)
   - 게시글에 이미지 첨부
   - 갤러리에서 이미지 선택
   - 카메라로 사진 촬영

3. **온보딩 화면** (`screens/AppIntroScreen.js`)
   - 초기 프로필 이미지 설정

### READ_MEDIA_VIDEO 사용 위치:
- 현재 코드에서는 동영상 기능을 사용하지 않음
- `expo-image-picker`와 `expo-media-library` 패키지가 Android 13+ (API 33+)에서 이 권한을 요구할 수 있음
- 향후 동영상 업로드 기능 추가 시 사용 예정
