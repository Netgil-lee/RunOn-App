# 구글 플레이 콘솔 데이터 사용 목적 선언 가이드

## 📱 RunOn 앱 데이터 사용 목적

### ✅ 선택해야 할 항목

다음 항목들을 모두 선택하세요:

1. ✅ **앱 기능 (App functionality)**
2. ✅ **애널리틱스 (Analytics)** - 이미 선택됨
3. ✅ **개발자 커뮤니케이션 (Developer communication)**
4. ✅ **사기 예방, 보안, 규정 준수 (Fraud prevention, security, and compliance)**
5. ✅ **맞춤설정 (Customization)**
6. ✅ **계정 관리 (Account management)**

### ❌ 선택하지 않을 항목

- ❌ **광고 또는 마케팅 (Advertising or marketing)** - 앱에서 광고를 표시하지 않음

---

## 📋 각 항목 선택 이유

### 1. ✅ 앱 기능 (App functionality)
**선택 이유:**
- Firebase 인증을 통한 사용자 로그인
- 기능 사용 설정 (프리미엄 기능 등)
- 사용자 인증 및 서비스 제공

**사용 예시:**
- 이메일 로그인
- 구글 로그인
- 앱 내 기능 활성화

---

### 2. ✅ 애널리틱스 (Analytics)
**선택 이유:**
- Firebase Analytics 사용
- 앱 사용 방식 및 성능 데이터 수집
- 버그 진단 및 성능 개선

**사용 예시:**
- 특정 기능 사용자 수 확인
- 앱 상태 모니터링
- 버그 및 비정상 종료 진단
- 향후 성능 개선

---

### 3. ✅ 개발자 커뮤니케이션 (Developer communication)
**선택 이유:**
- 푸시 알림을 통한 사용자 알림
- 모임 관련 알림 (취소, 새 참여자 등)
- 채팅 메시지 알림
- 커뮤니티 알림 (좋아요, 댓글)
- 날씨 경고 알림

**사용 예시:**
- 모임 취소 알림
- 새 참여자 알림
- 채팅 메시지 알림
- 좋아요/댓글 알림

---

### 4. ✅ 사기 예방, 보안, 규정 준수 (Fraud prevention, security, and compliance)
**선택 이유:**
- Firebase 인증을 통한 로그인 보안
- 사기 가능성 있는 활동 식별
- 사용자 인증 정보 확인

**사용 예시:**
- 로그인 시도 실패 모니터링
- 사용자 인증 및 보안
- 부적절한 활동 차단

---

### 5. ✅ 맞춤설정 (Customization)
**선택 이유:**
- 위치 기반 러닝 코스 추천
- 사용자 레벨/목표에 따른 추천
- 날씨 기반 러닝 추천

**사용 예시:**
- 사용자 위치를 기준으로 한 근처 러닝 코스 추천
- 사용자 레벨(초급/중급/고급)에 따른 러닝 페이스 추천
- 사용자 목표(습관/체중감량/5K 등)에 따른 추천
- 날씨 조건에 따른 러닝 추천

---

### 6. ✅ 계정 관리 (Account management)
**선택 이유:**
- 사용자 계정 생성
- 계정 정보 관리
- 프로필 정보 관리
- 로그인 및 인증

**사용 예시:**
- 회원가입 (계정 생성)
- 프로필 정보 추가/수정
- 로그인 및 인증
- 계정 설정 관리

---

### 7. ❌ 광고 또는 마케팅 (Advertising or marketing)
**선택하지 않는 이유:**
- 앱에서 광고를 표시하지 않음
- 마케팅 목적의 푸시 알림을 보내지 않음
- 광고 파트너와 데이터를 공유하지 않음

**참고:**
- Firebase Analytics는 사용하지만, 광고 목적이 아닌 앱 분석 목적입니다.
- 광고 ID는 Firebase SDK에서 자동으로 포함되지만, 실제로 광고를 표시하지 않으므로 이 항목은 선택하지 않습니다.

---

## 📝 "출시 오류 사용 중지" 체크박스

### ❌ 선택하지 마세요

**이유:**
- 매니페스트에 `com.google.android.gms.permission.AD_ID` 권한이 포함되어 있습니다.
- 이 권한을 제거하지 않을 예정이므로, 출시 오류를 숨길 필요가 없습니다.
- Google의 정책에 따라 올바르게 선언하는 것이 중요합니다.
