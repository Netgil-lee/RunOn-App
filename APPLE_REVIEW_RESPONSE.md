# Apple App Review - UGC 안전 기능 답변

## 질문 1: 부적절한 콘텐츠를 필터링하는 방법이 있습니까?

**답변: 예, 있습니다.**

RunOn 앱은 실시간 콘텐츠 필터링 시스템을 구현하고 있습니다.

### 구현 위치
- 게시글 작성 화면 (`PostCreateScreen`)
- 댓글 작성 화면 (`PostDetailScreen`)

### 작동 방식
1. **Level 1 (심각) - 즉시 차단**: 욕설, 성적 표현, 폭력, 혐오 표현 등이 포함된 콘텐츠는 게시 전에 자동으로 차단됩니다.
2. **Level 2 (경미) - 경고 후 선택**: 경미한 부적절한 표현은 경고 메시지와 함께 사용자에게 선택권을 제공합니다.

### 확인 방법
앱 실행 → 커뮤니티 탭 → 게시글 작성 → 부적절한 단어(예: "시발") 입력 시도 → 즉시 차단 메시지 확인

## 질문 2: 사용자가 부적절한 콘텐츠를 신고할 수 있는 메커니즘이 있습니까?

**답변: 예, 있습니다.**

RunOn 앱은 게시글, 댓글, 사용자에 대한 신고 기능을 제공합니다.

### 구현 위치
1. **게시글 신고**: 게시글 상세 화면 헤더 우측 경고 아이콘(⚠️)
2. **댓글 신고**: 댓글 길게 누르기 (Long Press)
3. **사용자 신고**: 사용자 프로필 화면

### 신고 사유
- 스팸/광고
- 부적절한 콘텐츠
- 혐오 표현
- 성적 콘텐츠
- 폭력/위협
- 기타

### 확인 방법
앱 실행 → 커뮤니티 탭 → 게시글 클릭 → 헤더 우측 경고 아이콘(⚠️) 클릭 → 신고 사유 선택 → 제출

---

## 질문 3: 사용자가 악의적인 사용자를 차단할 수 있는 메커니즘이 있습니까?

**답변: 예, 있습니다.**

RunOn 앱은 사용자 차단 기능을 제공합니다.

### 구현 위치
- 사용자 프로필 화면 (`ParticipantScreen`)
- 차단 관리 화면 (`BlacklistManagementScreen`)

### 작동 방식
1. 사용자 프로필 화면에서 "차단" 버튼 클릭
2. 양방향 차단 처리 (차단한 사용자와 차단당한 사용자 모두 서로를 차단)
3. 차단된 사용자와 함께 있는 모든 모임에서 자동 탈퇴
4. 차단된 사용자의 게시글/댓글이 표시되지 않음

### 차단 제한
- 최대 3명까지 차단 가능
- 차단 해제는 설정 화면에서 가능

### 확인 방법
앱 실행 → 모임 상세 화면 → 참여자 목록 → 사용자 프로필 클릭 → "차단" 버튼 클릭

## 질문 4: 개발자가 24시간 내에 부적절한 콘텐츠 신고에 대해 조치를 취합니까? (콘텐츠 제거 및 사용자 추방)

**답변: 예, 보장합니다.**

RunOn 앱은 24시간 내 신고 조치 시스템을 구현하고 있습니다.

### 자동 처리 시스템
1. **신고 접수**: 사용자가 신고 제출
2. **즉시 알림**: 관리자에게 이메일 알림 전송 (`dlrhdkgml12@gmail.com`)
3. **자동 처리 스케줄러**: 매시간 실행하여 24시간 경과한 미처리 신고를 자동으로 조치합니다. 이를 통해 신고 접수 후 최대 1시간 내에 조치가 완료되며, 24시간 내 조치를 보장합니다.
4. **자동 조치 내용**:
   - 게시글/댓글 자동 삭제
   - 신고된 사용자의 신고 횟수 증가
   - 신고 횟수 3회 이상 시 자동 계정 정지

### 관리자 수동 조치
- 신고 접수 이메일을 통해 Firestore에서 신고 확인
- 24시간 내 수동 조치 가능

### 조치 타임라인
- **0시간**: 신고 접수, 이메일 알림 전송
- **0-24시간**: 관리자 수동 조치 가능
- **24시간**: 자동 처리 스케줄러 실행, 미처리 신고 자동 조치
- **24시간 이후**: 신고 횟수 3회 이상 사용자 자동 계정 정지

### 확인 방법
1. 신고 제출
2. 관리자 이메일 확인
3. Firebase Console에서 신고 확인 (`reports` 컬렉션)
4. 24시간 후 자동 처리 확인 (`status: 'action_taken'` 확인)

### 테스트 계정 정보

**로그인 정보:**
- **휴대폰번호**: `010-0000-0000`
- **인증번호**: 아무 숫자나 입력 (예: `123456`)
- **SMS 인증**: 자동 우회됨

### 준비된 샘플 데이터

커뮤니티 탭에서 다음 샘플 데이터를 확인하실 수 있습니다:

#### 1. 게시판 (자유게시판 탭)
- ✅ **5개의 샘플 게시글**이 준비되어 있습니다:
  - 코스 추천 게시글 (한강공원 러닝 코스)
  - 러닝 팁 게시글 (초보자를 위한 팁)
  - 모임 후기 게시글 (러닝 모임 참여 후기)
  - 러닝 용품 추천 게시글 (러닝화 추천)
  - 자유 토크 게시글 (일상 대화)
- ✅ 각 게시글에는 **댓글과 좋아요**가 포함되어 있습니다
- ✅ 다양한 **카테고리** (자유 토크, 러닝 팁, 모임 후기, 코스 추천, 러닝 용품)가 포함되어 있습니다

#### 2. 채팅 (채팅 탭)
- ✅ **2개의 활성 채팅방**이 준비되어 있습니다:
  - "한강공원 아침 러닝 모임" 채팅방
  - "올림픽공원 저녁 러닝" 채팅방
- ✅ 각 채팅방에는 **6개의 샘플 메시지**가 포함되어 있습니다
- ✅ 실제 대화 형식의 메시지로 구성되어 있습니다

#### 3. 러닝 모임 (러닝모임 탭)
- ✅ **2개의 활성 러닝 모임**이 준비되어 있습니다:
  - 한강공원 아침 러닝 모임 (초급, 2일 후)
  - 올림픽공원 저녁 러닝 (중급, 5일 후)
- ✅ 각 모임에는 참여자 정보가 포함되어 있습니다

### 테스트 방법

1. **앱 실행** 후 로그인 화면에서:
   - 휴대폰번호: `010-0000-0000` 입력
   - 인증번호: `123456` (또는 아무 숫자) 입력
   - 자동으로 로그인됩니다

2. **커뮤니티 탭**으로 이동:
   - 하단 네비게이션 바에서 "커뮤니티" 탭 클릭

3. **각 탭 테스트**:
   - **자유게시판 탭**: 5개의 샘플 게시글 확인
     - 게시글 클릭하여 상세 보기
     - 댓글 작성 기능 테스트
     - 좋아요 기능 테스트
     - 신고 기능 테스트 (게시글 상세 화면 헤더 우측 ⚠️ 아이콘)
   
   - **채팅 탭**: 2개의 활성 채팅방 확인
     - 채팅방 클릭하여 메시지 확인
     - 메시지 전송 기능 테스트
     - 채팅방 참여자 정보 확인
   
   - **러닝모임 탭**: 2개의 활성 모임 확인
     - 모임 상세 정보 확인
     - 모임 참여 기능 테스트

### 샘플 데이터 생성

샘플 데이터는 다음 스크립트로 생성되었습니다:
